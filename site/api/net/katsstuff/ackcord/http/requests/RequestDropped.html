RequestDropped 

case class RequestDropped[Ctx](context: Ctx, uri: Uri, rawRoute: String) extends MaybeRequest[Nothing, Ctx] with FailedRequest[Ctx] with Product with Serializable

A request that was dropped before it entered the network, most likely because of timing out while waiting for ratelimits.
